claude-token-efficient is a single `CLAUDE.md` file you drop into your project root. Claude Code reads it automatically — no code changes, no configuration, no setup. Behavior changes immediately: shorter responses, no sycophantic filler, no unsolicited suggestions, ~63% fewer output tokens on heavy workflows.

## The problem

By default, Claude:

- Opens every response with "Sure!", "Great question!", "Absolutely!"
- Ends with "I hope this helps! Let me know if you need anything!"
- Uses em dashes, smart quotes, and Unicode that break parsers
- Restates your question before answering it
- Adds unsolicited suggestions beyond what you asked
- Over-engineers code with abstractions you never requested

**All of this wastes tokens. None of it adds value.**

## The fix

<Steps>
  <Step title="Download the file">
    ```bash
    curl -o CLAUDE.md https://raw.githubusercontent.com/drona23/claude-token-efficient/main/CLAUDE.md
    ```
  </Step>
  <Step title="Drop it in your project root">
    ```
    your-project/
    └── CLAUDE.md    ← one file, zero setup, no code changes
    ```
  </Step>
  <Step title="Start using Claude Code">
    Claude reads `CLAUDE.md` automatically. Responses become terse immediately.
  </Step>
</Steps>
